# Project: notice-service Test

## End-point: Create notice

### Method: PUT

> ```
> {{url}}/api/newNotice
> ```

### Body (**raw**)

```json
{
    "title": "타이틀1",
    "content": "타이틀1의 content입니다."
}
```

### Response: 200

```json
notice create ok
```

## End-point: Modify notice

### Method: POST

> ```
> {{url}}/api/notice/{{uuid}}
> ```

### Body (**raw**)

```json
{
    "title": "수정1",
    "content": "수정테스트중입니다다."
}
```

### Response: 200

```json
Notice updated successfully.
```
